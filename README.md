# nuclei-templates
some nuclei template for vulnerability discovery. 

## How to use
```bash
git clone https://github.com/madisec/nuclei-templates.git
cd nuclei-templates
echo {assets} | nuclei -t {TEMPLATE-PATH} -silent
```
## templates
- CVEs
- Injections vulnerability types
- XSS
- Open Redirect (oRedirect)
- SSRF
- and so on
